Rocket.Chat enables real-time conversations:

- Internally between colleagues  
- Across companies  
- With customers or citizens

# ğŸ’» Deploy Rocket.Chat

Different deployment methods exist depending on your needs:

- **Self-hosted**: Use Docker, Podman, or Kubernetes. Refer to the [Deployment Guide](https://docs.rocket.chat/docs/deploy-rocketchat) and [system requirements](https://docs.rocket.chat/docs/system-requirements).  
  Alternatively, use [Launchpad](https://docs.rocket.chat/docs/deploy-with-launchpad) for a quick Kubernetes setup.

- **Air-gapped workspace**: For isolated networks, see [air-gapped deployment](https://docs.rocket.chat/docs/rocketchat-air-gapped-deployment).

- **Cloud hosting**: Check out [Rocket.Chat Cloud](https://docs.rocket.chat/docs/rocketchat-cloud-hosting-service-level-agreement-sla) for managed hosting options.

- **Federation**: Enable [federation](https://docs.rocket.chat/docs/rocketchat-native-federation) for decentralized communication across networks.
