# web-ui-registration

This package contains the UI components and logic responsible for handling user registration in Rocket.Chat.

## Purpose
The `web-ui-registration` package provides the interface and supporting functionality required for users to create new accounts in Rocket.Chat.

## Features
- User registration form
- Input validation
- Integration with authentication services

## Usage
This package is used internally by the Rocket.Chat web client to render the registration flow.
