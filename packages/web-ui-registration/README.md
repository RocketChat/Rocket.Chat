# @rocket.chat/web-ui-registration

This package contains UI components and logic for the Rocket.Chat registration flow.

## Purpose

The main goal of this package is to provide a consistent and reusable set of components for the web-based registration process, ensuring a smooth onboarding experience for new users.

## Installation

This is internal to the Rocket.Chat monorepo. It is managed via Yarn Workspaces.

## Usage

Components from this package are used throughout the Rocket.Chat web client where registration or user onboarding is required.
